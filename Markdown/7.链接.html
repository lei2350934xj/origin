Markdown 支持两种形式的链接语法： 行内式和参考式两种形式 不管是哪一种，链接文字都是用 [方括号] 来标记。 要建立一个行内式的链接，只要在方块括号后面紧接着圆括号并插入网址链接即可，如果你还想要加上 链接的 title 文字，只要在网址后面，用双引号把 title 文字包起来即可，例如: This is [an example](http://example.com/ "Title") inline link. [This link](http://example.net/) has no title attribute. 会产生：

This is an example inline link.

This link has no title attribute.
