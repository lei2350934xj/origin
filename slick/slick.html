Features 特点

  • 完全响应式设计，随容器自动缩放。
  • 一个断点，一套设置。
  • 浏览器支持CSS3与否，全部功能不受影响。
  • 是否开启滑动，由你指定，
  • 同时支持鼠标拖动，
  • 以及自动轮播，
  • 更可使用键盘方向键切换。
  • 添加、移除、跳过指定幻灯片，
  • 自动播放，圆点、箭头跳转，执行回调函数，以及更多...

  slidesToScroll: 3

$('.js-add-slide').on('click', function() {
  slideIndex++;
  $('.add-remove').slick('slickAdd','<div><h3>' + slideIndex + '</h3></div>');
});

$('.js-remove-slide').on('click', function() {
  $('.add-remove').slick('slickRemove',slideIndex - 1);
  if (slideIndex !== 0){
    slideIndex--;
  }
});

var filtered = false;

$('.js-filter').on('click', function(){
  if (filtered === false) {
    $('.filtering').slick('slickFilter',':even');
    $(this).text('Unfilter Slides');
    filtered = true;
  } else {
    $('.filtering').slick('slickUnfilter');
    $(this).text('Filter Slides');
    filtered = false;
  }
});

Getting Started 使用步骤

定义 HTML 标记


<div class="your-class">
  <div>your content</div>
  <div>your content</div>
  <div>your content</div>
</div>

将 slick 文件夹移入你的项目

在 <head> 中添加 slick.css


<link rel="stylesheet" type="text/css" href="slick/slick.css"/>
// Add the new slick-theme.css if you want the default styling
<link rel="stylesheet" type="text/css" href="slick/slick-theme.css"/>

在jQuery之后，body 闭合标签之前添加 slick.js（需要 jQuery 1.7+）


<script type="text/javascript" src="//code.jquery.com/jquery-1.11.0.min.js"></script>
<script type="text/javascript" src="//code.jquery.com/jquery-migrate-1.2.1.min.js"></script>
<script type="text/javascript" src="slick/slick.min.js"></script>

在外部文件或内联脚本中初始化 slick


$(document).ready(function(){
  $('.your-class').slick({
    setting-name: setting-value
  });
});

完成上述步骤后，你的 HTML 文件应该类似于这样：


<html>
  <head>
  <title>My Now Amazing Webpage</title>
  <link rel="stylesheet" type="text/css" href="slick/slick.css"/>
  <link rel="stylesheet" type="text/css" href="slick/slick-theme.css"/>
  </head>
  <body>

  <div class="your-class">
    <div>your content</div>
    <div>your content</div>
    <div>your content</div>
  </div>

  <script type="text/javascript" src="//code.jquery.com/jquery-1.11.0.min.js"></script>
  <script type="text/javascript" src="//code.jquery.com/jquery-migrate-1.2.1.min.js"></script>
  <script type="text/javascript" src="slick/slick.min.js"></script>

  <script type="text/javascript">
    $(document).ready(function(){
      $('.your-class').slick({
        setting-name: setting-value
      });
    });
  </script>

  </body>
</html>

注意：强烈建议将初始化脚本放在外部文件中。

Settings 设置项

Setting Type Default Description
accessibility boolean true 启用 TAB 和方向键导航。
adaptiveHeight boolean false 让容器的高度随着幻灯片自身高度而变化。
autoplay boolean false 启用自动轮播。
autoplaySpeed int(ms) 3000 设置轮播速度，单位是毫秒。
arrows boolean true 显示左右箭头。
asNavFor string null 将当前轮播图与另一个轮播图关联（依据 class 或 id）。
appendArrows string $(element) 改变导航箭头的作用对象（选择器、htmlString、数组、元素、jQuery 对象）
appendDots string $(element) 改变导航圆点的作用对象（选择器、htmlString、数组、元素、jQuery 对象）
prevArrow string (html|jQuery selector) | object (DOM node|jQuery object) <button type="button" class="slick-prev">Previous</button> 让你自定义“前一个”箭头。
nextArrow string (html|jQuery selector) | object (DOM node|jQuery object) <button type="button" class="slick-next">Next</button> 让你自定义“后一个”箭头。
centerMode boolean false 启用“中央视图”，突出当前幻灯片，部分显示前一个、后一个幻灯片。与奇数的 slidesToShow 配合使用。
centerPadding string '50px' 设置“中央视图”启用时的左右 padding (px 或 %)。
cssEase string 'ease' 使用 CSS3 动画。
customPaging function n/a 自定义幻灯片标注（paging）模板。在“源码”中查看示例。
dots boolean false 显示圆点指示器。
dotsClass string 'slick-dots' 自定义圆点指示器的 class。
draggable boolean true 启用鼠标拖动。
fade boolean false 启用渐变效果。
focusOnSelect boolean false 启用点击时的聚焦效果。
easing string 'linear' 平滑 jQuery 动画。可以与实现平滑效果的库配合使用。
edgeFriction integer 0.15 禁用循环轮播时，尝试滑动第一张或最后一张幻灯片时的阻抗效果。
infinite boolean true 启用循环轮播。
initialSlide integer 0 从特定幻灯片开始轮播，默认为0，表示第一张幻灯片。
lazyLoad string 'ondemand' 设置懒加载。设为“ondemand”表示启用，“progressive”则预先加载所有幻灯片。
mobileFirst boolean false 使用“移动端优先”计算响应式表现的参数。
pauseOnFocus boolean true 聚焦时暂停自动播放。
pauseOnHover boolean true 悬浮时暂停自动播放。
pauseOnDotsHover boolean false 悬浮于导航圆点时暂停自动播放。
respondTo string 'window' 响应式对象的参照宽度。可以是“window”，“slider”，或“min”（前两者中的较小者）。
responsive object none 包含断点和 settings 对象（查看示例）。用于为不同屏幕宽度定义各自的 settings 对象。定义其中一个 settings 为“unslick”，则在对应的屏幕宽度上禁用 slick。
rows int 1 此项若设置为大于1的值，则会触发栅格模式。使用 slidesPerRow 设置每一行的幻灯片个数。
slide element '' 将要作为幻灯片的元素。
slidesPerRow int 1 设置当栅格模式被触发时，每一行的幻灯片个数。
slidesToShow int 1 第 # 个要显示的幻灯片。
slidesToScroll int 1 第 # 个要卷动的幻灯片。
speed int(ms) 300 滑动或淡出动画的速度。
swipe boolean true 用 swiping 效果。
swipeToSlide boolean false 让用户直接滑动至某个幻灯片，忽略 slidesToScroll 的值。
touchMove boolean true 启用触屏操作。
touchThreshold int 5 要让幻灯片发生切换，用户必须划过幻灯片宽度 1/touchThreshold 的距离。
useCSS boolean true 启用或禁用 CSS transition 效果。
useTransform boolean true 启用或禁用 CSS transform 效果。
variableWidth boolean false 幻灯片宽度可变。
vertical boolean false 垂直轮播模式。
verticalSwiping boolean false 用户垂直划过屏幕时切换幻灯片（默认为水平划过）。
rtl boolean false 调转轮播方向为自右向左。
waitForAnimate boolean true 动画进行时，忽略切换幻灯片的请求。
zIndex number 1000 设置幻灯片的 zIndex 参数，适用于 IE9 及之前版本的浏览器。

事件

自 1.4 版本起，回调方法（callback methods）已经被事件（events）替代。用法如下：


// On swipe event
$('.your-element').on('swipe', function(event, slick, direction){
  console.log(direction);
  // left
});

// On edge hit
$('.your-element').on('edge', function(event, slick, direction){
  console.log('edge was hit')
});

// On before slide change
$('.your-element').on('beforeChange', function(event, slick, currentSlide, nextSlide){
  console.log(nextSlide);
});
Event Arguments Description
afterChange slick, currentSlide 幻灯片切换之后触发。
beforeChange slick, currentSlide, nextSlide 幻灯片切换之前触发。
breakpoint event, slick, breakpoint 到达一个断点时触发。
destroy event, slick 幻灯片被销毁时，或 slick 被禁用时触发。
edge slick, direction 在非循环模式下，卷动到第一个或最后一个幻灯片边缘时触发。
init slick 初始化之后触发。
reInit slick 每重新初始化一次之后触发。
setPosition slick 位置或大小改变时触发。
swipe slick, direction 滑动或拖动时触发。
lazyLoaded event, slick, image, imageSource 图片懒加载时触发。
lazyLoadError event, slick, image, imageSource 图片加载失败后触发。

方法

从 1.4 版开始，调用一个方法是通过 .slick(method) 来实现的：


// Add a slide
$('.your-element').slick('slickAdd',"<div></div>");

// Get the current slide
var currentSlide = $('.your-element').slick('slickCurrentSlide');

这一新语法让你可以调用 slick 的内置方法：


// Manually refresh positioning of slick
$('.your-element').slick('setPosition');
Method Arguments Description
slickCurrentSlide none 返回当前幻灯片的序号
slickGoTo int : slide number, boolean: dont animate 根据给定序号定位到幻灯片
slickNext none 定位到后一张幻灯片
slickPrev none 定位到前一张幻灯片
slickPause none 暂停自动播放
slickPlay none 开始自动播放
slickAdd html or DOM object, index, addBefore 添加一张幻灯片。如果给定了序号，则在添加在对应位置（当设置了 addBefore 时，添加在前一个位置）。若没有给定序号，则添加到列表末尾（当设置了 addBefore 时，添加在开头）。接受 HTML 字符串或对象。
slickRemove index, removeBefore 根据给定序号删除一张幻灯片。若 removeBefore 设置为 true，则删除给定序号的前一张幻灯片（没有指定序号则删除第一张幻灯片）。如果 removeBefore 设置为 false，则删除给定序号的后一张幻灯片（没有指定序号则删除最后一张幻灯片）。
slickFilter Selector or Function 使用 jQuery 的 .filter() 过滤幻灯片
slickUnfilter index 移除先前设置过滤的幻灯片
slickGetOption option : string 获取指定 option 的值
slickSetOption option : string, value : depends on option, refresh : boolean 定义一个 option，赋值为 value。如果这是一个 UI 更新，设置 refresh 为 true。
unslick none 解构一个 slick
getSlick none 获取 slick 对象
